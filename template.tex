\documentclass{article} % Define a classe do documento

\usepackage[utf8]{inputenc} % Permite o uso de caracteres especiais
\usepackage[brazil]{babel} % Define o idioma do documento

\title{Título do Documento} % Título do documento
\author{Seu Nome} % Autor do documento
\date{\today} % Data atual

\begin{document} % Início do documento

\maketitle % Gera o título, autor e data

\section{Introdução} % Seção
Este é um exemplo básico de um documento em LaTeX. Aqui você pode escrever seu texto e incluir várias seções.

\section{Conteúdo} % Outra seção
Você pode usar \textbf{negrito}, \textit{itálico} e até criar listas:
\begin{itemize} % Lista não ordenada
    \item Item 1
    \item Item 2
    \item Item 3
\end{itemize}

\end{document} % Fim do documento
